## SQL 语法表达范式

[TBD: SQL 语法范式放到 SQL 手册]

在文档中，我们采取形式化的方式表示 SQL 语法规则：

- 注释（`--`、`/**/`）和分行用于辅助说明：

```sql
SELECT select_list  /*选择列表*/
FROM table          /*数据源*/
WHERE condition   -- 过滤条件
```

- 关键字大写，标识符（数据库名、表名、列名等）小写：

```sql
-- 语法表示：
CREATE DATABASE db_name;

-- 具体示例：
CREATE DATABASE power;
CREATE DATABASE atmosphere;
```

- `[option]`、`[option_1|option_2]` 表示可选项；`{option_1|option_2}` 表示必选项：

```sql
-- 语法表示：
SHOW {NORMAL|CHILD} [db_name.]TABLES [LIKE tb_name_wildchar]

-- 具体示例：
SHOW NORMAL TABLES;
SHOW CHILD power.TABLES LIKE "%e";
```

- `...` 表示前一个元素（token）可以被重复；`, ...` 表示重复元素之间需用 `,` 隔开：

```sql
-- 语法表示：
INSERT INTO table_name VALUES (value1, value2, ...) ...

-- 具体示例：
INSERT INTO table_1 VALUE (2,100,'Beijing') (3,95,'Shanghai');
INSERT INTO table_2 VALUE (1,2,3,4,5);
```

- 局部语法过于复杂，则用标识符代替，并在下文中展开，例如下文的 `select_list`：

```sql
-- 语法表示：
SELECT select_list FROM table_name

select_list:
    {[tb_name.]* | expr}, ...

-- 具体示例：
SELECT * FROM meters;
SELECT tb1.* FROM tb1;
SELECT ts,ABS(phase) FROM meters;
```

- 标识符的文本含义具有提示作用：

```sql
-- 语法表示：
SELECT * FROM table_name WHERE bool_expr;

-- 具体示例：
SELECT * FROM test.meters WHERE current > 5;
```
