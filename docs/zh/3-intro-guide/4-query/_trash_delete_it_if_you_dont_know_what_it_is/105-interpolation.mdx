---
title: 插值
description: 函数产值的使用方法
---

使用插值功能，可在指定时间段，按照指定方式插入数值。

```sql
SELECT _irowts, INTERP(current) FROM test.d1021 RANGE('2017-07-14 12:00:00.000','2017-07-15 12:00:00.000') EVERY(1h) FILL(LINEAR);
```

在上例中：

- `RANGE` 指定了插值的时间范围
- `EVERY` 指定了两个插值点之间的间隔为 1 小时
- `FILL` 指定了插值模式：使用前后两个已存在的数据点，进行线性插值（`LINEAR`）
- `INTERP` 指定了被插值的变量为 `current`
- 插值点的时间戳由 `RANGE` 和 `EVERY` 规定，需使用 `_irowts` 来查看该时间戳
